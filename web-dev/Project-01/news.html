"Elden Ring Nightreign" is a new co-op spin-off with martial arts combat.

FromSoftware has announced "Elden Ring Nightreign," a cooperative multiplayer spin-off set in a parallel universe to the original game. The game features fast-paced, martial arts-inspired combat and allows up to three players to team up as Nightfarers. Players can choose from eight unique characters, each with distinct abilities, to battle monsters and face bosses. A network test for Nightreign is scheduled for February 2025, with registrations opening on January 10, 2025. The full game is set to release later in 2025 on PlayStation 5 and Xbox Series X|S platforms.
